```rust
//! Bandwidth estimation surfaces for the kit.
//!
//! str0m 0.18 internally runs a libWebRTC GoogCC (`src/bwe/`) — trendline delay
//! estimator + MLE loss controller + leaky-bucket pacer — but its output is
//! only visible via `str0m::Event::EgressBitrateEstimate`. The kit translates
//! that event into `Propagated::BandwidthEstimate` so downstream
//! (the caller's room logic or a LayerSelector trait implementation) can
//! adapt forwarding without importing str0m types.

use std::time::Duration;

/// Aggregated egress bandwidth estimate for a peer's outgoing stream.
///
/// Units: bits per second. Rust convention — `0` is legal (meaning the
/// estimator has observed total network failure or has no data yet).
#[derive(Debug, Clone, Copy, PartialEq)]
pub struct BandwidthEstimate {
    /// Current estimate in bits per second.
    pub bps: u64,
    /// Minimum of the estimator's bounds at the time of emission.
    pub min_bps: u64,
    /// Maximum of the estimator's bounds at the time of emission.
    pub max_bps: u64,
    /// Age of this estimate since estimator startup.
    pub estimator_uptime: Duration,
}

impl BandwidthEstimate {
    /// Zero-bandwidth fallback for tests.
    pub fn zero() -> Self {
        Self {
            bps: 0,
            min_bps: 0,
            max_bps: 0,
            estimator_uptime: Duration::ZERO,
        }
    }
}
```

## Task 5 — Buffer pool for forwarding

Firezone-borrow from `rust/libs/connlib/bufferpool`. Self-contained pool for byte-vecs to avoid allocation in hot forward path.

**Files:**
- Create: `src/buffer_pool.rs`
- Modify: `src/registry/mod.rs` (use pool in forward), `src/lib.rs`

- [ ] **Step 1: Write `src/buffer_pool.rs`**

Port the essential pattern from firezone `bufferpool/lib.rs`. Essentials:

```rust
//! Byte-buffer pool to reduce allocation in the forward path.
//!
//! Pattern: `BufferPool::acquire()` returns a `Buffer` with Drop-return.

use std::sync::{Arc, Mutex};

#[derive(Clone)]
pub struct BufferPool {
    inner: Arc<Mutex<Vec<Vec<u8>>>>,
    initial_cap: usize,
}

impl BufferPool {
    pub fn new(initial_cap: usize) -> Self {
        Self {
            inner: Arc::new(Mutex::new(Vec::new())),
            initial_cap,
        }
    }

    pub fn acquire(&self) -> Buffer {
        let buf = self
            .inner
            .lock()
            .expect("poisoned")
            .pop()
            .unwrap_or_else(|| Vec::with_capacity(self.initial_cap));
        Buffer {
            bytes: Some(buf),
            pool: self.clone(),
        }
    }
}

pub struct Buffer {
    bytes: Option<Vec<u8>>,
    pool: BufferPool,
}

impl std::ops::Deref for Buffer {
    type Target = Vec<u8>;
    fn deref(&self) -> &Self::Target {
        self.bytes.as_ref().expect("buffer taken")
    }
}

impl std::ops::DerefMut for Buffer {
    fn deref_mut(&mut self) -> &mut Self::Target {
        self.bytes.as_mut().expect("buffer taken")
    }
}

impl Drop for Buffer {
    fn drop(&mut self) {
        if let Some(mut bytes) = self.bytes.take() {
            bytes.clear();
            self.pool.inner.lock().expect("poisoned").push(bytes);
        }
    }
}
```

- [ ] **Step 2: Integrate in `Registry::forward` or similar hot path**

Replace `let buf = Vec::with_capacity(MTU)` with `let mut buf = self.pool.acquire()`. Use `buf.extend_from_slice(...)` then let Drop return it.

- [ ] **Step 3: Tests**

    let pool = BufferPool::new(1500);
    let ptr_a = {
        let buf = pool.acquire();

    };
